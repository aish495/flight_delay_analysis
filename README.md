Flight Delay Analysis using R

Overview
This project performs Exploratory Data Analysis (EDA) on the NYC Flights 2013 dataset using R. The analysis focuses on understanding patterns in departure and arrival delays across airlines, airports, and months.

The project applies statistical analysis and data visualization techniques to identify trends, skewness, outliers, and operational insights from flight delay data.

Objectives
Analyze departure and arrival delays
Compare delay patterns across airlines and airports
Study monthly trends in flight delays
Identify skewness and outliers in delay distributions
Generate business insights through data analysis

Tools & Technologies
R
dplyr
ggplot2
Exploratory Data Analysis (EDA)
Statistical Analysis
Data Visualization

Key Insights
Flight delays show positively skewed distributions with significant outliers
Departure delays are generally higher than arrival delays
Certain airlines recorded consistently higher delays
Peak delays were observed during mid-year months
Some flights were able to recover delays during airtime
